# F1 Race Predictor Dashboard 🏎️

A Machine Learning-powered web application designed to predict F1 race outcomes. Built with Flask, this tool leverages historical data to provide insights on race performance.

## Features
* **Predictive Analysis:** Uses trained Scikit-learn models to estimate race finishes.
* **Data-Driven:** Analyzes factors including Grid Position, Championship Points, Pit Strategy, and Weather Conditions.
* **Interactive Dashboard:** A clean, user-friendly web interface built with Flask.

## Tech Stack
* **Language:** Python 3.x
* **Web Framework:** Flask
* **Machine Learning:** Scikit-learn, Pandas
* **Data Source:** FastF1 API
* **Deployment:** Render
